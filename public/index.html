Guide & Tutorial to this Express App for "Users"

This app supports the following functions:

  • List of All Users Available - /users (GET)
  • Detailed Information About a Specific User - /users/id (GET)
  • To Create a New User - /users (POST) with first_name, last_name, employment_status and age
  • To Update an Existing User - /users/id (PUT) with first_name, last_name, employment_status and age
  • To Delete an Existing/Specific User - /users/id (DELETE)
